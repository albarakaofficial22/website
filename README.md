# Al Baraka Supermarket Website

## Description
A beautiful, modern website for Al Baraka Supermarket featuring a fresh vegetable-inspired color scheme. The website showcases the store's products, services, and contact information with a clean, responsive design.

## Features
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Vegetable-Inspired Colors**: Fresh, natural color palette inspired by produce
- **Smooth Animations**: Interactive elements with smooth transitions
- **Contact Integration**: WhatsApp and email contact options
- **Product Showcase**: Six main product categories displayed
- **Services Section**: Highlighting key store services

## Color Scheme
The website uses a vegetable-inspired color palette:
- Leafy Green (#4A7C59) - Primary brand color
- Tomato Red (#D64545) - Accent color
- Carrot Orange (#FF8C42) - Call-to-action elements
- Cucumber Green (#8FBC8F) - Secondary green
- Bell Pepper Yellow (#FFD166) - Highlights

## How to Deploy on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com) and sign in
2. Click the "+" icon in the top right and select "New repository"
3. Name your repository (e.g., "albaraka-supermarket")
4. Make sure it's set to "Public"
5. Don't initialize with README (we already have one)
6. Click "Create repository"

### Step 2: Upload Files
1. On your new repository page, click "uploading an existing file"
2. Drag and drop all website files (or click "choose your files")
3. Make sure to upload:
   - `index.html`
   - `styles.css`
   - `script.js`
   - The entire `images` folder with all SVG files
4. Add a commit message like "Initial website upload"
5. Click "Commit changes"

### Step 3: Enable GitHub Pages
1. In your repository, go to "Settings" (top menu)
2. Scroll down to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Under "Branch", select "main" (or "master")
5. Select "/ (root)" as the folder
6. Click "Save"

### Step 4: Access Your Website
1. After a few minutes, your site will be available at:
   `https://[your-username].github.io/[repository-name]/`
2. GitHub will show you the URL at the top of the Pages settings
